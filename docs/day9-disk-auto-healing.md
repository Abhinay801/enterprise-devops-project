# Day 9 - Disk Cleanup Auto-Healing

## Features

- Disk usage monitoring
- Threshold-based cleanup
- Automatic deletion of old logs
- Cleanup history tracking
- Report generation

## Commands Used

- df
- find
- wc
- delete

## Output Locations

auto-healing/disk-cleanup/reports/
auto-healing/disk-cleanup/logs/

| Command        | Purpose                          |
| -------------- | -------------------------------- |
| `find`         | Search for files and directories |
| `find -delete` | Delete matching files            |
| `wc -l`        | Count lines/files                |
| `df`           | Check disk usage                 |
