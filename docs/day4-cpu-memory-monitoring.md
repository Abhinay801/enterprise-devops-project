# Day 4 - CPU & Memory Monitoring Module

## Objective

Monitor CPU and Memory usage and generate alerts.

## Features

- CPU monitoring
- Memory monitoring
- Threshold-based alerts
- Timestamp-based reports

## Thresholds

CPU Threshold: 80%
Memory Threshold: 80%

## Commands Used

- top
- free
- awk
- date

## Output Location

monitoring/cpu/reports/
| Command | Purpose                        |
| ------- | ------------------------------ |
| `top`   | Display real-time system usage |
| `free`  | Show memory usage              |
| `awk`   | Process and filter text        |
| `date`  | Generate timestamps            |
