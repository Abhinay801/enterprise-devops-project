# Day 7 - Enterprise Server Health Report

## Features

- CPU monitoring
- Memory monitoring
- Disk monitoring
- Server information collection
- Uptime monitoring
- Top process analysis
- Overall health status generation

## Commands Used

- hostname
- uptime
- top
- free
- df
- ps

## Report Location

monitoring/server-health/reports/


| Command     | Purpose                    |
| ----------- | -------------------------- |
| `hostname`  | Displays system hostname   |
| `uptime -p` | Shows system uptime        |
| `top`       | Displays CPU usage         |
| `free`      | Displays memory usage      |
| `df`        | Displays disk usage        |
| `ps`        | Displays running processes |
