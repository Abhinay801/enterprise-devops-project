# Day 6 - Log Monitoring Module

## Features

- Detects ERROR logs
- Detects WARNING logs
- Detects CRITICAL logs
- Counts log occurrences
- Generates timestamp-based reports

## Commands Used

- grep
- grep -c
- grep -E
- date

## Report Location

monitoring/logs/reports/
| Command   | Purpose                  |
| --------- | ------------------------ |
| `grep`    | Search text patterns     |
| `grep -c` | Count matching lines     |
| `grep -E` | Search multiple patterns |
| `date`    | Generate timestamps      |
